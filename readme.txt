"Modal Portfolio" permet d'afficher des galeries d'images filtr&eacute;es avec titre et description dans une modale à l'aide d'un shortcode (avec options). ("Modal Portfolio" displays image galleries filtered with title and description in a modal using some shortcodes)

== Description ==

<p><em>En fran&ccedil;ais</em></p>
<p><strong>Modal Portfolio</strong> fonctionne en anglais et en fran&ccedil;ais<br/>
<strong>Modal Portfolio</strong> utilise le shortcode [modal-portfolio] (avec ou sans options) pour afficher des galeries d'images avec titre et description dans une modale. Beaucoup de param&egrave;tres sont disponibles pour optimiser l'affichage ainsi que des filtres d'affichage par cat&eacute;gorie.<br/><br/>
La principale particularit&eacute; de Modal Portfolio est de g&eacute;rer plusieurs cat&eacute;gories de portfolio avec des shortcodes diff&eacute;rents et personnalisables. Ainsi, vous pouvez avoir plusieurs portfolios selon les cat&eacute;gories choisies.</p>

<p><em>For English people</em></p>
<p><strong>Modal Portfolio</strong> works in English and in French<br/>
<strong>Modal Portfolio</strong> uses the shortcode [modal-portfolio] (with or without options) for displaying image galleries with title and description in a modal. Many settings are available to optimize the display and display filters by category<br/><br/>
The main feature of Modal Portfolio is to manage several categories of portfolio with different customizable shortcodes . Thus, you can have multiple portfolios depending on the chosen categories.</p>

== Installation ==

<p><em>En fran&ccedil;ais</em></p>
<p>1. Recopier le contenu de l'archive dans le r&eacute;pertoire des extensions (wp-content/plugins) et activer dans le panneau des extensions du backoffice de Wordpress.<br/>
2. R&eacute;gler les param&egrave;tres dans les options de l'extension.<br/>
3. Utiliser le shortcode [modal-portfolio] pour afficher les r&eacute;f&eacute;rences (options disponibles dans la documentation du plugin).</p>

<p><em>For English people</em></p>
<p>1. Copy the content of the archive in "wp-content/plugins" and activate the plugin.<br/>
2. Adjust the settings in the options.<br/>
3. Use the shortcode [modal-portfolio] to show the items (several options availables in the documentation of this plugin).</p>
